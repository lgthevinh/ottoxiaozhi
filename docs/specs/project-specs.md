# Otto Xiaozhi

## Server 
- **Language**: Python
- **Framework**: FastAPI

### Audio Streaming
- **Format**: PCM (Pulse Code Modulation)
- **Sample Rate**: 16 kHz
- **Channels**: Mono (1 channel)
- **Bit Depth**: 16 bits per sample
- **Frame Duration**: 20 ms (320 samples per frame, 640 bytes per frame)
- **Streaming Method**: Websocket

- **Audio Handler**: Handle audio conversion from WAV to PCM and manage audio streaming sessions.
